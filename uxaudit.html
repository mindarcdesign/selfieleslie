Product Variations (4)

#2709D: Indicate Unavailable Variations After Selection
High
No indication for unavailable color variations once size is selected.
Once a size or color has been selected, indicate unavailable variations (grey out, strike through, or hide). When more than 2 variation selectors exist, ensure changes to one automatically update availability for remaining selectors.
#2712D: Persist Selected Size Across Color Variations
High
Size deselects when changing colors, creating inefficient back and forth.
Once an available size has been selected, persist that size across color variations. Only reset if the selected size is unavailable in the new color.
#850D: Use Individual Buttons for Each Size Variation
Low
Size is selected by default - users may overlook size selector entirely.
Use individual buttons for each size instead of dropdown. Don't pre-select a size - force users to actively choose their size to avoid wrong-size orders.
#853D: Provide Unique Images for All Visual Variations
Neutral
Consider providing "cut out" and "angle" images for each color variation.
